# 🚁 无人机飞控智能体 - UAV Agent 自我介绍

## 👋 你好!我是谁?

我是**无人机飞控智能体(UAV Agent)**,一个专为无人机飞行控制和任务执行而生的AI智能助手。我能够理解你的自然语言指令,并将其转化为精确的飞控操作,让无人机操控变得像对话一样简单。

---

## 🎯 我的核心能力

### 1️⃣ **智能飞行控制**

我可以帮你完成各种飞行任务,从简单的起飞降落到复杂的空间定位飞行:

#### ✈️ 基础飞行动作
- **一键起飞** - 指定高度和目标点,自动完成起飞流程
- **飞向目标点** - 控制无人机飞向指定的经纬度坐标
- **一键返航** - 安全返回起飞点
- **紧急悬停** - 遇到危险时立即停止飞行
- **航线暂停/恢复** - 灵活控制任务执行节奏

#### 🎮 精准操控
- **杆量控制** - 通过虚拟摇杆实现毫米级精准飞行
- **DRC指令飞行** - 进入专业指令飞行控制模式
- **目标点更新** - 飞行过程中动态调整目的地
- **控制权抢夺** - 多用户环境下获取飞行控制权

**示例指令**:
```
"飞到软件园三期F06栋上空100米"
"向前飞50米,然后悬停"
"立即返航"
"暂停当前任务"
```

### 2️⃣ **相机与拍摄控制**

我支持多种拍摄模式,满足巡查、取证、测绘等场景需求:

#### 📸 拍照功能
- **单张照片拍摄** - 指定相机拍照取证
- **全景照片** - 通过多张照片拼接实现全景覆盖

#### 🎥 录像功能
- **开始录像** - 记录飞行全过程
- **停止录像** - 结束视频录制
- **多相机切换** - 支持可见光、红外、夜视多种镜头

#### 🔭 云台控制
- **看向目标点** - 自动调整相机朝向指定位置
- **画面拖动** - 平滑调整拍摄角度

- **云台重置** - 一键恢复默认姿态

**示例指令**:
```

"切换到红外镜头"
```

- 异常情况下主动通知
- 完成后总结执行结果

## 🎓 最佳实践

### ✅ 推荐做法
1. **明确目标** - 说清楚要飞到哪里、做什么
2. **分步执行** - 复杂任务分解为多个步骤
3. **实时监控** - 开启直播观察飞行过程
4. **预留电量** - 返航电量至少保留20%

### ❌ 避免事项
1. **模糊指令** - "飞过去看看"(没说去哪里)
2. **越权操作** - 未抢夺控制权就发送指令
3. **忽略状态** - 设备离线时强行操作
4. **超限飞行** - 超出安全高度或距离限制

## 🆘 常见问题

**Q: 我能同时控制多架无人机吗?**  
A: 当前版本一次控制一架无人机,多机协同功能开发中。

**Q: 语音识别准确率低怎么办?**  
A: 建议使用普通话,避免方言和背景噪音。也支持文字输入。

**Q: 飞行过程中能修改目标点吗?**  
A: 可以,使用"更新目标点"指令即可动态调整。

**Q: 为什么有时操作失败?**  
A: 可能原因:设备离线、权限不足、机场忙碌、电量不足。我会告知具体原因。

**Q: 能自动规划航线吗?**  
A: 目前支持指点飞行和航线执行,智能航线规划功能即将上线。

## 📝 复杂任务示例

### 🎯 多站点巡查完整流程

**场景**: 从软件园三期F06栋起飞,依次巡查F09、F08、F05栋,最后返航

**完整指令**:
```
请执行以下巡查任务:

1. 从软件园三期F06栋起飞,起飞高度50米,目标高度100米
2. 到达后先拍照取证,然后开始录像记录全过程
3. 云台看向东南方向,变焦到5倍焦距
4. 打开红外相机,切换到白热模式,设置区域测温
5. 打开探照灯,亮度80%
6. 飞到软件园三期F09栋,到达后拍3张不同角度的照片
7. 上升30米到130米高度,然后向左转45度
8. 用TTS播报"无人机巡查中,请注意安全"
9. 飞到软件园三期F08栋,框选画面中心区域进行变焦
10. 重置云台,切换到高清直播让我实时监控
11. 前进100米,再右转90度
12. 飞到软件园三期F05栋,到达后停止录像,保存视频
13. 关闭红外相机,切回可见光模式
14. 喊话器循环播放预设的警告音频
15. 最后执行一键返航,返航高度150米
16. 返航途中保持标清直播节省带宽
```

**自然语言简化版**:
```
"帮我执行一个巡查任务:

先从F06栋起飞到100米高,到了之后拍张照然后开始录像。
把相机转向东南方,放大5倍,打开红外用白热模式测温,探照灯也打开。

然后飞到F09栋,从不同角度拍3张照片。接着再升高到130米,向左转45度。
用语音播报一下'无人机巡查中请注意安全'。

飞到F08栋,框选画面中间放大看看,云台复位,给我开高清直播。
往前飞100米,右转90度。

到F05栋后停止录像,关掉红外切回普通模式,循环播放警告音频。
最后设置150米返航高度返航,路上直播切到标清省流量。"
```

### 🔧 关键API调用示例

#### 1. 一键起飞(F06栋)
```bash

  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
```
